Seadragon - An OpenSeadragon Gem for Ruby

The Seadragon Gem provides the necessary methods for generating DZI (Deep Zoom Image) tiles for use with OpenSeadragon. The Gem is bundled with the OpenSeadragon Javascript and image files and provides a simple helper method for dislaying zoomable images on your webpage.
